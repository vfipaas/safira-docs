## `safira-cli dockerhub:credential:set`

Configure Docker Hub credential

```
USAGE
  $ safira-cli dockerhub:credential:set

OPTIONS
  -p, --password=password  password
  -u, --username=username  username
  --help                   Show CLI help.

EXAMPLES
  safira-cli dockerhub:credential:set
  safira-cli dockerhub:credential:set --username=username --password=password
```
