## Instaling Okteto
Install okteto following these steps:

```
$ curl https://get.okteto.com -sSfL | sh
```
![Okteto](/img/screenshot/Okteto/Okteto1.png)

## Setting up Okteto

You can setup an remote develop enviroment with [Okteto Cloud](https://cloud.okteto.com/) or any Kubernetes cluster with Okteto CLI, in this guide we will use Okteto Cloud.
The first thing you need to do it's configure the Okteto CLI to use Okteto Cloud, run this command for it:

```
$ okteto context use https://cloud.okteto.com
```

![Okteto](/img/screenshot/Okteto/Okteto2.png)

You can confirm that your CLI is setup running this command:

```
$ okteto context list
```

![Okteto](/img/screenshot/Okteto/Okteto3.png)


## Starting your API

Generating your API with Safira-CLI grants the creation of an [Okteto manifest](https://www.okteto.com/docs/reference/manifest) and we will be using it to deploy or API to the develop enviroment running this command:
```
$ okteto up
```

![Okteto](/img/screenshot/Okteto/Okteto4.png)
