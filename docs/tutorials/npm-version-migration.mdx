:::info
O **safira-cli** a partir da versão **0.7.0** não será mais distribuído pelo npm.
:::

## Removendo versão npm do seu sistema

```bash
npm uninstall -g @vfipaas/safira-cli
```
<br/>

:::caution

Caso o comando **safira-cli** ainda esteja sendo reconhecido pelo sistema, tente limpar o cache do npm.

:::

    
```bash 
npm cache clean -f
```
