Primeiro Projeto com Safira.

Iniciando o projeto com o Quarkus.​

  1. Crie uma pasta que você queira que seu projeto seja criado.

  2. Na pasta criada abra um terminal de linha de comando da sua preferência (Bash, ZSH, e etc.).

  1. Para criar um projeto use o comando:
safira-cli project:new PROJECT-NAME

Substitua PROJECT-NAME pelo nome do projeto desejado ou use "." para criar o projeto na pasta atual.

  1. Escolha se deseja usar um arquivo OpenAPI ou um arquivo do Insomnia Workspace.

  1. Diga o caminho de onde está seu arquivo OpenAPI(yaml/json) com as specs de seu projeto.

  1. Escreva uma descrição para seu Projeto.

  1. Escolha uma porta que seu projeto irá usar, por padrão o Safira usa a porta 8080.

  1. Escolha se deseja criar um mapeamento dos endpoints para o Kong.

  1. Qual framework deseja trabalhar.

  1. A nomenclatura dos pacotes java do projeto.

  1. O Gerenciador de pacotes que deseja utilizar.

  1. Pronto o seu projeto Hello World está criado.

A API usada para o passo a passo está disponível no repositório da NHS Digital.​
