Okta

O que é o Okta?​

A ferramenta Okta é usada para nós autenticarmos o acesso ao nosso Dev Portal.

Começando com Okta​

  1. Acesse o site do Okta Developer e crie uma conta a partir de seu e-mail.

  1. Para criar o autenticador de acesso do seu Dev Portal siga o caminho Applications > Applications > Create App Integration.

Em Create App Integration selecione as seguintes opções:

  • OIDC - OpenID Connect
  • Web Application

Nas configurações da Aplicação marque os seguintes itens:

Em Grant type :

  • Refresh Token

Em Assignments:

  • Skip group assignment for now

Pode finalizar a criação da Aplicação.

Após a criação da aplicação guarde as seguintes informações : Client ID, Client Secret e URL (Na URL delete o texto "-admin") de seu Portal Okta para usar na configuração de seu Dev Portal!

Criação de grupos.​

No menu lateral de seu Portal Okta vá em Directory > Groups > Add Group, acesse a página do grupo criado clicando em seu nome e no menu selecione Applications > Assign Application e adicione à lista a Aplicação criada.
