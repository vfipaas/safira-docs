Instalação Linux

Instalando no Debian/Ubuntu​

Exemplo de instalação no Debian/Ubuntu.

Pré-requisitos​

  • docker

  • node_gyp

  • python_v3.6+

  • nodejs14+

  • make

  • cmake

  • g++

  • jq

  • curl

  • java8+

  • maven

Utilize o safira-doctor para verificar se os pré-requisitos estão instalados​

npx @vfipaas/safira-doctor
note

Caso não possua o npx instalado, siga essa instrução para o node e volte novamente para este passo.

Se obteve uma resposta como o exemplo acima siga para Instalação do Safira-CLI. Caso obtenha uma resposta diferente da mostrada no exemplo acima, por favor refira-se as instruções abaixo para a instalação da(s) dependência(s) que não estão compativeis com os pré-requisitos.

Dependências​

sudo apt update
sudo apt install zip unzip curl -y

Instalando Docker​

Mais sobre docker

sudo apt-get install apt-transport-https ca-certificates curl gnupg lsb-release -y && \
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg && \
echo \
  "deb [arch=amd64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu \
  $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null && \
sudo apt-get update && \
sudo apt-get install docker-ce docker-ce-cli containerd.io -y && \
sudo usermod -aG docker $USER && \
newgrp docker

Instalando NodeJS 16 usando nvm​

Mais sobre o nvm

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
source ~/.bashrc
nvm install 16

Instalando Java usando o SdkMan​

Mais sobre o SdkMan

curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
source ~/.bashrc 
sdk install java 11.0.12-open

Instalando demais dependências​

sudo apt update
sudo apt install node-gyp make cmake g++ jq curl maven
